# TypeScript Projects

Welcome to the TypeScript projects repository. This repository contains several TypeScript exercises designed to enhance your understanding of TypeScript features and practices. Each task is organized into separate directories.

## Task Descriptions

### Task 0: Creating an Interface for a Student

- **Objective**: Create an interface for a student and render a table with student information.
- **Files**:
  - `task_0/js/main.ts`: Contains TypeScript code for creating the `Student` interface, initializing student data, and rendering it in a table.
- **Requirements**:
  - Use TypeScript and Vanilla JavaScript to render student data.
  - Ensure no TypeScript errors when running Webpack.

### Task 1: Building and Extending Interfaces

#### Task 1.1: Build a Teacher Interface

- **Objective**: Create a `Teacher` interface with specific attributes and dynamic property support.
- **Files**:
  - `task_1/js/main.ts`: Contains TypeScript code for creating the `Teacher` interface.

#### Task 1.2: Extending the Teacher Class

- **Objective**: Extend the `Teacher` interface to create a `Directors` interface with additional attributes.
- **Files**:
  - `task_1/js/main.ts`: Contains TypeScript code for creating the `Directors` interface.

#### Task 1.3: Printing Teachers

- **Objective**: Write a `printTeacher` function and interface for formatting teacher names.
- **Files**:
  - `task_1/js/main.ts`: Contains TypeScript code for the `printTeacher` function and related interface.

### Task 2: Working with Classes and Advanced Types

#### Task 2.1: Writing a Class

- **Objective**: Create a `StudentClass` with specific methods and interfaces.
- **Files**:
  - `task_1/js/main.ts`: Contains TypeScript code for the `StudentClass`.

#### Task 2.2: Advanced Types Part 1

- **Objective**: Create interfaces and classes for `Director` and `Teacher`, and a function to create employees based on salary.
- **Files**:
  - `task_2/js/main.ts`: Contains TypeScript code for `Director`, `Teacher`, and `createEmployee` function.

### Task 3: Ambient Namespaces

- **Objective**: Work with ambient namespaces and declare types for a CRUD module.
- **Files**:
  - `task_3/js/main.ts`: Contains TypeScript code to use the ambient module.
  - `task_3/js/interface.ts`: Contains TypeScript types and interfaces.
  - `task_3/js/crud.d.ts`: Contains ambient type declarations for the `crud.js` module.

### Task 4: Namespace & Declaration Merging

- **Objective**: Use namespaces and declaration merging to create and manage `Teacher` and `Subject` classes.
- **Files**:
  - `task_4/js/subjects/Cpp.ts`, `Java.ts`, `React.ts`, `Subject.ts`, `Teacher.ts`: Contains TypeScript code for managing subjects and teachers with namespaces.
  - `task_4/js/main.ts`: Contains TypeScript code to create and use `Cpp`, `Java`, and `React` subjects.

### Task 5: Brand Convention & Nominal Typing

- **Objective**: Implement brand convention for `MajorCredits` and `MinorCredits` interfaces and create functions to sum credits.
- **Files**:
  - `task_5/js/main.ts`: Contains TypeScript code for `MajorCredits`, `MinorCredits`, and functions to sum credits.
